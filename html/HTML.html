HTML 元素语法

HTML 元素以开始标签起始
HTML 元素以结束标签终止
元素的内容是开始标签与结束标签之间的内容
某些 HTML 元素具有空内容（empty content）
空元素在开始标签中进行关闭（以开始标签的结束而结束）
大多数 HTML 元素可拥有属性






















CSS样式

一. 行内样式
当特殊的样式需要应用到个别元素时，就可以使用行内样式。 使用行内样式的方法是在相关的标签中使用样式属性。样式属性可以包含任何 CSS 属性。
二. 内部样式
当单个文件需要特别样式时，就可以使用内部样式表。
三. 链入外部样式
当样式需要被应用到很多页面的时候，链入外部样式表将是理想的选择。
四. 导入外部样式表
等于将CSS代码导入并存在网页内部






















CSS 元素选择器

一. 标记选择器
基本语法：
标记名{属性1：值1；属性2：值2；……}
二. 类选择器
基本语法：
类.标记名{属性1：值1；属性2：值2；……}
三. id选择器
基本语法：
标记名#id名{属性1：值1；属性2：值2；……}
四. 通用选择器
基本语法：
*{属性1：值1；属性2：值2；……}
